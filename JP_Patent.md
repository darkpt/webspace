### 📋 階段總覽表

| 階段 | 程序名稱 | 主要活動 | 關鍵期限 | 法條依據 |
|:---:|---------|---------|---------|---------|
| **🅐** | **申請前準備與遞交** | • 製備申請文件<br/>• 提交願書<br/>• 繳納申請費 | - | 第36條、第38條 |

| **🅒** | **實體審查** | • 專利要件審查<br/>• 拒絕理由通知<br/>• 答辯與修正 | • 拒絕理由通知60日內答辯 | 第49條-第52條 |
| **🅓** | **核准與登錄** | • 繳納特許料<br/>• 設定登錄<br/>• 發行公報 | • 核准查定後30日內繳費 | 第66條、第108條 |
| **🅔** | **核准後程序** | • 異議申立<br/>• 無效審判<br/>• 訂正審判 | • 異議：核准後6個月<br/>• 無效/訂正：無期限 | 第113條、第123條、第126條 |

### 🔄 程序分流說明

#### 第二階段分流點
```
申請日起3年內 ──→ 請求實體審查 ──→ 進入第三階段
                 └→ 未請求 ──→ 申請案視為撤回
```

#### 第三階段分流點
```
實體審查結果 ──→ 核准 ──→ 進入第四階段
             └→ 駁回 ──→ 駁回查定不服審判 ──→ 撤銷 ──→ 重新審查
                                         └→ 維持 ──→ 確定駁回
```

#### 第五階段程序選項
```
核准後6個月內 ──→ 異議申立（任何人）
專利權存續期間 ──→ 無效審判（利害關係人）
專利權存續期間 ──→ 訂正審判（專利權人）
延長登錄專利 ──→ 延長登錄無效審判（利害關係人）
```

### 📌 重要規定與注意事項

#### ▶ 申請人資格（第33條）
- 發明人或承繼人有權申請專利
- 共同發明須共同申請（第38條）

#### ▶ 先願主義（第39條）
- 同一發明有多件申請時，最先申請者取得專利權
- 同日申請時，由申請人協議決定

#### ▶ 單一性要件（第37條）
- 一案一發明原則
- 符合發明單一性之一群發明可列於同一申請案

### 📌 實體審查判斷基準

#### 1️⃣ 產業利用性（第29條第1項前段）
- 發明必須在產業上可利用

#### 2️⃣ 新穎性（第29條第1項）

**不具新穎性之情形（第29條第1項各款）：**
```
├─ 公開已知之發明
├─ 公開實施之發明
└─ 刊物已揭示之發明
```

#### 3️⃣ 進步性（第29條第2項）
- 發明對所屬技術領域具通常知識者非顯而易見

#### 4️⃣ 不予專利之發明（第32條）

| 類型 | 說明 |
|-----|------|
| 公序良俗違反 | 有害公共秩序或善良風俗者 |
| 特定方法 | 特定醫療方法等 |

#### 5️⃣ 明細書記載要件（第36條第3項-第6項）

| 要件 | 內容 | 條項 |
|-----|------|------|
| **實施可能性** | 發明可據以實施 | 第36條第4項第1款 |
| **請求範圍支持性** | 權利範圍有明細書支持 | 第36條第6項第1款 |
| **明確性** | 權利範圍界定明確 | 第36條第6項第2款 |

#### ▶ 補正限制（第17條之2第3項-第5項）

**補正三原則**：
1. **範圍限制**：不得超出願書最初記載事項範圍
2. **時機限制**：僅於特定時點可補正
3. **目的限制**：僅限特定補正目的

**允許之補正目的**（第17條之2第5項）：
```
├─ 請求項之刪除
├─ 權利範圍之減縮
├─ 誤記之訂正
└─ 不明確記載之釋明
```

- 審決謄本送達後**30日內**
