📄 Google 文件處理工具

自動處理 Google Sheets 與 Google Docs 之間的文字替換

📋 使用說明

Google Sheets 格式：

• A欄：要被替換的原始文字

• B欄：要添加的補充內容

Google Docs 要求：

• 文件中必須包含 【實施方式】 和 【符號說明】 段落標記

如何取得 ID：

• 從 URL 中提取 ID 部分

• 範例：https://docs.google.com/spreadsheets/d/1Ra6nCigMRMw3SajFgaYXinpFLqhmZ-Vtab1WJCGm1kY/edit

請輸入 Google Sheets 的文件 ID（從 URL 中的 /d/ 後面部分提取）
請輸入 Google Docs 的文件 ID（從 URL 中的 /d/ 後面部分提取）
